| Method      | Example            | Description                   |
| ----------- | ------------------ | ----------------------------- |
| `pop(key)`  | `car.pop("model")` | Removes and returns the value |
| `del`       | `del car["brand"]` | Deletes key-value pair        |
| `clear()`   | `car.clear()`      | Empties the dictionary        |
| `popitem()` | `car.popitem()`    | Removes last inserted item    |
